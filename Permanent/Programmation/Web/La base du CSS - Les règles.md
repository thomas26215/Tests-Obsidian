**CSS :**
Une règle CSS se compose :
- D'un sélecteur (Exemple p)
- D'un ensemble de propriétés valuées
**Exemple :**
```CSS
p{
	font-weight: bold;
	color: red;
	margin: 10px;
	padding: 10px;
}
```

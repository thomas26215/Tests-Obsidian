[[Définition - Notion de condition]]

Voici la forme générale d'une condition en Java :
```Java
if(condition1){
	//traitement  si réalisé
}else if(condition 1){
	//traitement si réalisé
}else{
	//traitement si aucune condition réalisé
}
```
